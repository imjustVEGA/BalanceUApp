Carpeta para imágenes de entrenamientos

Coloca aquí las imágenes de los entrenamientos con los siguientes nombres:

- imagen_torso.png (o .jpg, .jpeg, .webp)
- imagen_piernas.png
- imagen_fuerza.png
- imagen_cardio.png
- imagen_no_equipment.png
- imagen_pesas.png

Formatos soportados: PNG, JPG, JPEG, WebP

Las imágenes se cargarán automáticamente como fondo de las tarjetas de entrenamiento.
