ESTRUCTURA DE IMÁGENES PARA EJERCICIOS
========================================

Cada ejercicio tiene su propia carpeta dentro de: app/src/main/assets/ejercicios/

ESTRUCTURA:
-----------
ejercicios/
  ├── flexiones/
  │   └── imagen.png (o imagen.jpg, imagen.jpeg, imagen.webp)
  ├── sentadillas/
  │   └── imagen.png
  ├── plancha/
  │   └── imagen.png
  ├── yoga_montana/
  │   └── imagen.png
  └── [nombre_ejercicio]/
      └── imagen.png

NOMBRES DE ARCHIVOS ACEPTADOS:
-------------------------------
Para cada ejercicio, puedes usar cualquiera de estos nombres:
- imagen.png
- image.png
- foto.png
- photo.png
- [nombre_ejercicio].png

FORMATOS ACEPTADOS:
-------------------
- .png
- .jpg
- .jpeg
- .webp

EJEMPLOS:
--------
Para el ejercicio "Flexiones":
  ejercicios/flexiones/imagen.png

Para el ejercicio "Postura de la montaña":
  ejercicios/postura_de_la_montana/imagen.png

Para el ejercicio "Press de banca":
  ejercicios/press_de_banca/imagen.png

NOTAS:
------
- Los nombres de las carpetas se generan automáticamente desde el nombre del ejercicio
- Se eliminan acentos y espacios se convierten en guiones bajos
- La imagen se ajustará automáticamente para que quepa completamente en el espacio disponible
